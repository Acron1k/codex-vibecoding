# Personal CRM Experience Prototype

Интерфейс на React + Vite, иллюстрирующий 5 ключевых страниц персональной CRM-системы:

- **Командный центр** — агрегированный дашборд с метриками, фокус-проектами, таймлайном и привычками.
- **Проекты и задачи** — спринты, канбан-поток, контроль точек и ресурсный план.
- **CRM-процессы** — воронка, грядущие коммуникации, сервис-деск и автоматизации.
- **Ритмы и привычки** — трекер рутин, недельный план, ритуалы и ретроспектива.
- **Финансы и аналитика** — бюджеты, кэшфлоу, ближайшие события и финансовое самочувствие.

## Локальный запуск

```bash
cd frontend
npm install
npm run dev
```

После запуска приложение будет доступно по адресу http://localhost:5173. Страницы переключаются через боковое меню.

## Скрипты

- `npm run lint` — проверка ESLint.
- `npm run build` — сборка production-версии.

Все данные в макете статичные и служат для демонстрации UX/визуального языка будущей системы.
