# VibeCoding Personal CRM UI

Набросок интерфейса персональной CRM с контекстами «Основная работа», «Интеграторство» и «Личное». Сейчас реализованы 5 ключевых экранов с демо-данными, чтобы визуализировать будущее приложение и обсудить следующие итерации.

## Страницы
- **Пульс дня** — обзор задач, таймлайна и ключевых метрик по всем контекстам.
- **Проектный центр** — Kanban по инициативам основной работы и интеграторства, контрольные точки.
- **Привычки и ритмы** — стрики, ритуалы и эксперименты для поддержки энергии.
- **Финансы** — недельный cashflow, показатели бюджета и расшифровка расходов.
- **Знания и ассистенты** — плейбуки, автоматизации и бэклог идей.

## Стек
- [Vite](https://vitejs.dev/) + [React](https://react.dev/) + [TypeScript](https://www.typescriptlang.org/)
- [React Router](https://reactrouter.com/) для навигации по страницам
- Кастомные стили на CSS с поддержкой тёмной темы, стеклянных панелей и адаптивности

## Скрипты
```bash
npm install          # установка зависимостей
npm run dev          # запуск дев-сервера
npm run build        # сборка production-версии (tsc + Vite)
npm run lint         # проверка eslint
```
