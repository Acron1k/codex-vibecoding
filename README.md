# VibeOS — персональная CRM-концепция

Фронтенд-прототип ключевых экранов персональной CRM для управления рабочими проектами, подработкой и личным развитием. Интерфейс построен вокруг единого пространства с боковой навигацией, динамическими карточками и стеклянными панелями.

## Что реализовано
- **Обзор** — сводка по контекстам, таймлайн дня, инсайты и текущие привычки.
- **Проекты** — карточки ключевых инициатив, визуал доски спринтов и очередь автоматизаций.
- **Привычки** — архитектура ритуалов, пульс недели и список экспериментов.
- **Финансы** — показатели месяца, прогноз выручки, бюджеты и ближайшие движения.

Каждый экран наполнен шаблонными данными, демонстрирующими предполагаемый UX и визуальный стиль.

## Технологический стек
- [React](https://react.dev/) + [TypeScript](https://www.typescriptlang.org/)
- [Vite](https://vitejs.dev/) для сборки
- [React Router](https://reactrouter.com/) для клиентской навигации
- [lucide-react](https://lucide.dev/) для иконок

## Запуск и разработка
```bash
npm install     # установка зависимостей
npm run dev     # локальный режим разработки на http://localhost:5173
npm run lint    # проверка eslint
npm run build   # сборка production-бандла
```
